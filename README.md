# 基金实时估值工具

一个基于前端静态页面的基金持仓管理工具，支持：

- 添加基金持仓
- 实时刷新基金估值
- 查看持仓盈亏和收益率
- 记录每日盈亏快照
- 展示收益曲线与持仓分布

## 在线访问

如果仓库已启用 GitHub Pages，项目页地址通常为：

`https://tristantian.github.io/test_html/`

## 本地运行

在项目目录启动一个静态服务：

```bash
python3 -m http.server 8000
```

然后在浏览器访问：

`http://127.0.0.1:8000`

## 使用方式

1. 打开“添加基金”
2. 输入 6 位基金代码并查询
3. 填写买入均价、持有份额、买入日期
4. 添加到持仓后，在“我的持仓”中查看估值和盈亏
5. 点击“记录今日盈亏”积累历史数据
6. 在“收益曲线”和“盈亏记录”中查看趋势

## 技术说明

- 前端：HTML / CSS / JavaScript
- 图表：Plotly
- 数据存储：`localStorage`
- 数据来源：页面运行时通过基金接口拉取估值数据，仅供参考，不构成投资建议
